Perform a principal components analysis to obtain per-cell coordinates in low-dimensional space.

Parameters
matThe input log-expression matrix, with features in rows and cells in columns.
numberNumber of PCs to obtain. Must be less than the smaller dimension of mat.
[in]subsetOffset to an input array of uint8_ts of length mat.nrow(), indicating which features should be used for the PCA.
scaleWhether to standardize rows in mat to unit variance. If true, all rows in mat are assumed to have non-zero variance.
pcsOffset to an output array of doubles of length number * mat.ncol().
prop_varOffset to an output array of doubles of length number.
Returns
pcs is filled with the PC coordinates in a column-major manner. prop_var is filled with the percentage of variance explained by each successive PC.
