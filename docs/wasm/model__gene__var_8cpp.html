Model the variance of the log-expression values for each gene, accounting for the mean-variance trend.

Parameters
matAn input log-expression matrix containing features in rows and cells in columns.
spanThe span of the LOWESS smoother for fitting the mean-variance trend.
use_blocksWhether or not to compute the statistics within each block. If false, the number of blocks is assumed to be 1, otherwise it is determined from the maximum value of blocks.
blocksIf use_blocks = true, offset to an array of int32_ts with ncells elements, containing the block assignment for each cell. Block IDs should be consecutive and 0-based. If use_blocks = false, this value is ignored.
[out]meansOffset to a 2D array of double with number of rows and columns equal to mat.nrow() and the number of blocks, respectively. Each column contains the mean log-expression of each feature in the corresponding block.
[out]variancesOffset to a 2D array of double with number of rows and columns equal to mat.nrow() and the number of blocks, respectively. Each column contains the variance of log-expression of each feature in the corresponding block.
[out]fittedOffset to a 2D array of double with number of rows and columns equal to mat.nrow() and the number of blocks, respectively. Each column contains the fitted value of the trend of each feature in the corresponding block.
[out]residualsOffset to a 2D array of double with number of rows and columns equal to mat.nrow() and the number of blocks, respectively. Each column contains the residual from the trend of each feature in the corresponding block.
Returns
The arrays in means, variances, fitted and residuals are filled. If use_block = true, statistics are computed separately for the cells in each block.
